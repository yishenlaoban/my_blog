# 5.SQL技巧

LOCATE(expr1,expr2):查找第一个参数在第二个参数中的位置，不存在是返回0；

CONCAT(expr1,expr2):连接两个参数，如果expr1值为“12”，expr2值为“3”，拼接后得到的结果值为“123”；

IFNULL(expr1,expr2):如果expr1不为null，则返回expr1，否则返回expr2；
